Mobit 文档介绍

欢迎来到 Mobit 文档，这里提供了使用 GraphQL 查询区块链数据的指南。本指南假设你已经具备 GraphQL 的基础知识，包括查询、变更和类型的概念。如果你是 GraphQL 新手，建议先阅读 GraphQL 官方文档。

GraphQL 端点​

Mobit 提供两个主要的 GraphQL 端点，用于查询区块链数据：

  1. Unistate CKB 测试端点：

    • URL: https://unistate-ckb-test.unistate.io/v1/graphql
    • 用途: 适用于 CKB 测试网络上的测试和开发。

  2. CKB 图谱主网端点：

    • URL: https://ckb-graph.unistate.io/v1/graphql
    • 用途: 适用于与主网 CKB 网络进行生产环境的应用。

这些端点允许你执行复杂的查询，检索各种类型的区块链数据，包括地址、区块、交易、代币、NFT 等。

文档结构​

本文档分为几个部分，每个部分重点介绍 Mobit GraphQL 模式中的不同核心类型及其相关字段。以下是关键部分的简要概述：

  1. 核心类型

    • 包括 addresses、block_height 和 transaction_outputs_status 等基本类型概述。
    • 提供检索和分析这些核心类型的常见查询示例。

  2. RGBPP 类型

    • 详细介绍 RGB++ 系统，该系统实现了比特币和 CKB 之间的跨链资产转移。
    • 解释 RGB++ 锁定脚本（Locks）和解锁脚本（Unlocks），以及如何管理资产状态。

  3. Spore 类型

    • 深入介绍 Spore NFT 系统，包括 spores 和 clusters。
    • 提供查询单个 Spore、集群及其相关操作的示例。

  4. 代币类型

    • 介绍 XUDT（用户自定义代币）系统，包括 token_info 和 xudt_cell。
    • 提供检索代币信息、余额和交易历史的查询示例。

每个部分都包含详细的 GraphQL 类型描述、字段说明以及使用 LightGraphQLPlayground 组件的实际查询示例。这些示例将帮助你构建和执行针对 Mobit GraphQL 端点的查询。

如何开始​

要开始从 Mobit GraphQL 端点查询数据，请按照以下步骤操作：

  1. 访问 GraphQL Playground: 使用浏览器导航到其中一个提供的端点。
  2. 编写你的查询: 使用 GraphQL Playground 接口编写和执行你的查询。
  3. 探索示例: 参考文档中的示例，获取灵感和指导。

通过使用 GraphQL 的强大功能，你可以高效地检索和分析区块链数据，从而构建强大的应用程序和集成 Mobit 生态系统。

如果你遇到任何问题或有疑问，欢迎联系我们的技术支持团队或社区论坛寻求帮助。祝你查询愉快！
